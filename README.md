# Software Requirements Specification (SRS) & System testing

## Project Title
CHAECK


## Project Description
독서는 오랫동안 개인의 지식 확장과 사유의 도구로 자리 잡아왔지만, 디지털 시대에 접어들면서 독서 경험을 기록하고 공유하는 방식이 점점 다양해지고 있습니다. 최근에는 단순히 책을 읽는 것에서 나아가, 독서 기록을 시각화하거나 다른 독자와 의견을 나누며 독서 경험을 확장하려는 수요가 꾸준히 증가하고 있습니다. 본 프로젝트는 책 추천과 서평 작성을 중심으로 한 웹 서비스 “CHAECK”을 기획하게 되었습니다. 사용자는 도서를 검색하고, 서평을 작성하거나 공유하며, 독서 기록과 AI를 기반으로 맞춤형 책을 추천받을 수 있습니다. 이를 통해 사용자들은 체계적인 독서 습관을 형성하고, 비슷한 관심사를 가진 사람들과 책에 대한 생각을 나누며, 더욱 풍부한 독서 경험을 누릴 수 있을 것입니다.
