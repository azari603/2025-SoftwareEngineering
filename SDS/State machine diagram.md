1\. **로그인 회원가입 단계**  
  사용자는 CHAECK의 모든 기능을 사용하기 위해서 로그인해야한다. 로그인이 필요한 기능을 사용하려고 하면 로그인 페이지가 나타난다. 또는 화면 오른쪽 상단의 로그인 버튼을 누르면 로그인 페이지로 이동한다. 로그인 종류로는 소셜로그인, 일반로그인이 있다. 소셜로그인은 소셜아이콘을 눌러 외부 플랫폼에서 인증을 진행한다. 일반로그인은 아이디와 비밀번호를 입력하면 서버가 db에서 정보를 대조하여 인증여부를 결정한다. 회원이 아닌 사용자는 회원가입을 해야한다. 사용자는 화면 오른쪽 상단의 회원가입 버튼을 누르거나 로그인 페이지에서 회원가입 버튼을 눌러 회원가입을 진행할 수 있다. 회원가입 종류로는 소셜 회원가입, 일반 회원가입이 있다. 소셜 회원가입은 소셜아이콘을 눌러 외부플랫폼에서 회원가입을 진행한다. 일반회원가입은 아이디, 이메일, 비밀번호, 비밀번호 확인을 입력하고 형식과 중복여부를 마치면 회원가입을 완료할 수 있다.

2\. **사이트 메인화면**  
  메인화면은 사용자가 처음 사이트에 접속할 시 보여지는 화면이다. 로그인한 유저와 로그인하지 않은 유저 모두 사용이 가능하며, 로그인 한 유저는 해당 유저의 독서 기록을 기반으로 추천 도서 목록과 팔로잉 하는 유저의 서평 목록인 팔로잉 피드가 보여지고 로그인하지 않은 유저에게는 인기있는 도서 목록과 오늘의 추천 서평 목록을 보여준다. 상단에는 책 기록하기(서평 작성 바로가기) 버튼과 서평 둘러보기 버튼이 있어 바로 해당 페이지로 이동할 수 있다.

3\. **프로필 확인 및 수정 기능**  
  사용자는 프로필 화면에서 이달의 목표를 볼 수 있고, 사용자가 지정한 별점별로 도서 목록을 확인 할 수 있다. 공개된 서평을 확인할 수 있다. 계정 설정 화면으로 이동하여 프로필 설정 버튼을 클릭 후 프로필을 수정할 수 있고 비밀번호와 이메일을 변경할 수 있는 계정 설정 버튼이 존재한다. 다른 사용자의 프로필을 확인하고 있는 경우 현재 사용자가 해당 사용자를 팔로잉 중이면 팔로우 취소 버튼이, 팔로우 중이지 않으면 팔로우 버튼이 활성화된다.

4\. **책 검색 및 책 정보 확인 기능**  
  검색 기능은 사용자가 책 제목이나 저자 이름으로 찾고자 하는 도서를 검색할 수 있는 기능이다. 메인 홈이나 헤더의 검색창에 제목이나 저자를 입력 후 엔터나 검색 버튼을 클릭 할 시 검색 결과 화면으로 넘어간다. 검색 결과에서 도서를 선택할 시 책의 상세정보 확인 페이지로 넘어가며 해당 페이지에서 해당 책의 정보와 리뷰, 추천 도서를 확인할 수 있다.

5\. **서평 작성 기능**  
  사용자가 서평작성 기능을 통해 읽은 책의 서평을 작성할 수 있다. 서평 작성을 책 상세보기 페이지에서 서평 쓰기 버튼을 통해 시작한 경우 책 선택 화면을 생략한다. 메인페이지에서 책 기록하기(서평 쓰기) 버튼, 헤더의 글쓰기 버튼을 통해서 시작한 경우에는 책 선택 화면에서 책을 선택한 뒤 작성화면으로 넘어간다다. 서평 작성을 할때에는 책을 읽기 시작한 날짜, 다 읽은 날짜, 공개/비공개 여부, 제목, 본문을 필수로 입력해야한다. 작성된 서평은 공개/비공개 여부에 따라 사람들에게 공개된다.

6\. **서평 확인 및 상호작용 기능**  
  사용자는 서평 확인/상호작용 기능을 통해 공개된 서평의 정보를 상세하게 확인할 수 있다. 사용자는 모든 서평에 대해서 좋아요, 댓글 작성이 가능하고, 다른 이용자의 서평인 경우 해당 이용자에 대한 팔로우, 팔로우 취소가 가능하다. 자신의 서평을 확인하는 경우, 자신의 서평에 대해 삭제, 수정을 수행할 수 있다. 

7\. **나의 기록 확인 기능**  
  기록확인 기능은 사용자 자신이 선택한 책들에 한해서 기록할 수 있는 기능이다. 나의 서재에서 읽고있는 책, 읽고싶은 책, 다 읽은 책 탭을 선택하여 도서 상태에 따른 목록을 확인할 수 있다. 나의 기록 \-\> 나의 서평에서 사용자가 작성한 서평 목록 혹은 좋아요를 클릭한 목록을 알 수 있다. 나의 기록 \-\> 독서 통계 페이지에서는 상단의 목표 설정 버튼을 클릭해 이달의 목표를 설정할 수 있고, 또는 이 달의 목표 달성률 그래프를 통해 얼마나 목표를 달성하였는지 확인할 수 있다. 사용자가 읽은 독서 기록을 통해 자주 읽은 저자의 통계가 집계되며, 특정 저자를 클릭하면 해당 저자의 책들 중 사용자가 읽은 책들의 목록을 알 수 있다. 또한 사용자의 도서별 평점을 그래프로 시각화하여 보여준다.

8\. **책BTI 기능**  
  로그인한 사용자에 한해 제공하는 기능이다. 사용자는 상단바에서 책BTI 메뉴를 클릭하여 테스트 화면으로 진입할 수 있다. 테스트 하러가기 버튼을 누르면 본격적으로 테스트가 진행된다. 다양한 질문에 응답하여 테스트를 완료 후 테스트 결과 화면으로 진입하며 해당 페이지에서 사용자는 자신의 독서 성향과 그에 맞는 책을 추천받을 수 있다.

9\. **로그아웃**  
  사용자는 상단바에서 로그아웃을 할 수 있다. 로그아웃 후에도 검색 기능, 책 정보 조회, 서평 조회, 프로필 조회 등의 기능은 이용할 수 있다.
