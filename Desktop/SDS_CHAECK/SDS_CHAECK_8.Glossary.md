# Glossary

| 이름 | 설명 |
|---|---|
| 팔로워/팔로잉 | 사용자끼리의 상호작용으로 팔로잉한 사용자의 서평을 쉽게 확인할 수 있다. |
| 책BTI | 책과 MBTI의 합성어, 사용자의 독서 성격을 파악할 수 있다. |
| 도서 상태 | 사용자가 도서 별로 상태를 지정할 수 있다. 읽고있는 책, 읽고싶은 책, 다 읽은 책으로 분류된다. |
| 나의 기록 | 나의 서재(도서 상태별 목록), 나의 서평(내가 작성한 서평 목록), 독서 통계를 확인 할 수 있는 페이지로, 각 탭을 통해 확인할 수 있다. |
| 아이디/닉네임 | 아이디는 로그인시 사용하는 정보(중복불가), 닉네임이란 상대방에게 표시되는 사용자(나)의 이름(중복가능) |
| 도서 정보 시스템 (naver 도서 api) | 아직 DB에 존재하지 않는 도서 정보를 받아오기 위한 api이다. 본 프로젝트에서는 네이버에서 제공해주는 naver 도서 api를 사용한다. |
| 프로필 설정 | 프로필 화면에서 사진과 닉네임 및 소개를 변경할 수 있다. |
| 계정 정보 설정 | 이메일과 비밀번호를 변경할 수 있다. |
| 둘러보기 | 다른 사용자의 서평 목록을 확인할 수 있다. 로그인한 사용자에 한해 팔로우 중인 사용자들의 서평 목록을 모아 볼 수 있다. |
